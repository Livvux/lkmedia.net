## 1. Installation und erste Einrichtung

### Schritt 1: Plugin installieren

1. **WordPress Admin** → **Plugins** → **Neues Plugin hinzufügen**
2. Suchen Sie nach "**Rank Math SEO**"
3. **Installieren** und **Aktivieren**

### Schritt 2: Setup Wizard starten

Nach der Aktivierung startet automatisch der Setup Wizard. Falls nicht, gehen Sie zu **Dashboard** → **Rank Math** → **Setup Wizard**.

#### Setup Wizard Schritt-für-Schritt:

**1. Willkommen & Verbindung (30 Sekunden)**

- Rank Math Account erstellen/verbinden
- Zugang zu Premium-Features und Updates

**2. Website-Typ auswählen**

```
Blog/Magazine    → Persönlicher Blog oder Nachrichten-Site
Business/Small   → Unternehmens-Website
E-Commerce      → Online-Shop (WooCommerce)
Portfolio       → Portfolio/Agentur-Website
```

**3. Bestehende SEO-Daten importieren**

- **Von Yoast SEO**: Alle Meta-Daten, Redirections, Einstellungen
- **Von All in One SEO**: Vollständige Migration
- **Von SEOPress**: Nahtloser Wechsel
- **Von RankMath**: Bei Neuinstallation

**4. Google Services verbinden**

- **Search Console**: Performance-Daten direkt in WordPress
- **Analytics**: Traffic-Analyse und Keyword-Insights

**5. Sitemap-Konfiguration**

- XML-Sitemap automatisch aktivieren
- An Google Search Console übermitteln

**✅ Setup Wizard Ergebnis:**

- Grundkonfiguration abgeschlossen
- Google Services verbunden
- Sitemap aktiviert und übermittelt
- Bereit für erweiterte Konfiguration

## 3. Allgemeine Einstellungen optimieren

**Navigation:** Rank Math → **General Settings**

### Links-Konfiguration

#### External Links

```php
✅ Strip Category Base    - Entfernt /category/ aus URLs
✅ Attachment Redirect    - Leitet Anhang-Seiten zur Attachment weiter
✅ Redirect Attachment    - Verhindert Duplicate Content

External Links:
✅ Open in New Tab       - Externe Links in neuem Tab
✅ Add nofollow         - Automatisches nofollow für externe Links
✅ Add noreferrer       - Datenschutz-optimiert
✅ Add sponsored        - Für gesponserte Links (FTC-konform)
```

**Warum diese Einstellungen:**

- **nofollow**: Verhindert Link-Juice Verlust an externe Seiten
- **noreferrer**: Schützt Referrer-Daten (DSGVO-konform)
- **sponsored**: Kennzeichnet bezahlte Links für Google

### Images-Konfiguration

```php
Images Settings:
✅ Add Missing Alt Text      - Generiert Alt-Text aus Dateiname
✅ Add Missing Title Text    - Erstellt Title-Attribute
✅ Open Attachment Page     - Für Bilder-SEO (optional)

Alt Text Format: %filename%
Title Text Format: %filename%
```

**Best Practice für Alt-Text:**

```php
// Automatisch generiert
Dateiname: wordpress-seo-tipps-2025.jpg
Alt-Text: "WordPress SEO Tipps 2025"

// Manual optimization
Alt-Text: "WordPress SEO Tipps für bessere Google Rankings 2025"
```

```php
✅ Enable Breadcrumbs
Separator: → (oder / oder >)
```

**Navigation:** Rank Math → **Sitemap Settings**

### Grundkonfiguration

#### Allgemeine Sitemap-Einstellungen

```php
✅ Enable Sitemap
Sitemap URL: https://ihre-domain.de/sitemap_index.xml

✅ Include Images in Sitemap
✅ Include Featured Image
✅ Remove /sitemap.xml redirect (falls vorhanden)
```

### Post Types konfigurieren

#### Posts (Blog-Artikel)

```php
✅ Include in Sitemap
Links per Sitemap: 200
Priority: 0.9 (hoch, da Content regelmäßig aktualisiert)
```

#### Pages (Seiten)

```php
✅ Include in Sitemap
Priority: 0.8 (hoch, da wichtige Business-Seiten)
```

#### Custom Post Types

```php
Produkte (WooCommerce): ✅ Include
Portfolio-Items: ✅ Include
Testimonials: ❌ Exclude (meist nicht suchrelevant)
```

### Taxonomies (Kategorien/Tags)

#### Kategorien

```php
✅ Include in Sitemap
Priority: 0.7
```

#### Tags

```php
❌ Include in Sitemap (verhindert Thin Content Issues)
```

### Erweiterte Sitemap-Einstellungen

#### Exclude/Include Rules

```php
Exclude Posts/Pages:
- Login-Seiten
- Thank-you-Seiten
- Admin-Seiten
- Test-Seiten

Author Sitemap: ❌ (meist unnötig, außer bei Multi-Author-Blogs)
Date Archive: ❌ (SEO-Value meist gering)
```

### Sitemap-Übermittlung

#### Automatische Übermittlung

```php
✅ Google Search Console - Automatisch
✅ Bing Webmaster Tools - Falls eingerichtet

Manuelle Prüfung:
1. Besuchen Sie: https://ihre-domain.de/sitemap_index.xml
2. Prüfen Sie alle Sub-Sitemaps
3. Validieren Sie in Google Search Console
```

#### Sitemap-Index Struktur

```xml
<?xml version="1.0" encoding="UTF-8"?>
<sitemapindex>
    <sitemap>
        <loc>https://ihre-domain.de/post-sitemap.xml</loc>
        <lastmod>2025-01-19T10:30:00+01:00</lastmod>
    </sitemap>
    <sitemap>
        <loc>https://ihre-domain.de/page-sitemap.xml</loc>
        <lastmod>2025-01-19T10:30:00+01:00</lastmod>
    </sitemap>
    <sitemap>
        <loc>https://ihre-domain.de/category-sitemap.xml</loc>
        <lastmod>2025-01-19T10:30:00+01:00</lastmod>
    </sitemap>
</sitemapindex>
```

## 6. Schema Markup & Rich Snippets

![Schema Markup Einstellungen](/images/blog/rankmath-guide/rankmath-schema-settings.webp)

**Navigation:** Rank Math → **Schema Templates**

### Grundlagen: Warum Schema wichtig ist

Schema Markup hilft Google dabei, Ihren Content besser zu verstehen und als Rich Snippet anzuzeigen:

- **Höhere CTR**: Rich Snippets fallen mehr auf
- **Bessere Rankings**: Google bevorzugt strukturierte Daten
- **Voice Search**: Optimierung für Sprachsuche
- **Featured Snippets**: Höhere Chance auf Position 0

```json
  "name": "LK Media",
  "url": "https://lkmedia.de",
  "logo": "https://lkmedia.de/logo.png",
  "description": "Webdesign & SEO Agentur in Karlsruhe",

    "@type": "PostalAddress",
    "streetAddress": "Felchenstraße 21",
    "addressLocality": "Rastatt",
    "postalCode": "76437",
    "addressCountry": "DE"

  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+49-721-123456",
    "contactType": "customer service",
    "availableLanguage": "German"
  }
}
```

## 14. Troubleshooting & Häufige Probleme

### Problem: SEO-Einstellungen werden nicht angezeigt

#### Lösung:

```php
1. Cache leeren (Plugin + Server)
2. Rank Math → Status & Tools → Clear SEO Cache
3. robots.txt prüfen (keine Blockierung)
4. Theme-Kompatibilität testen
```

### Problem: Schema Markup Fehler

#### Google Search Console Errors beheben:

```php
Häufige Schema-Fehler:
❌ Missing required field 'image'
✅ Lösung: Default-Bild in Schema-Einstellungen

❌ Invalid URL format
✅ Lösung: Absolute URLs verwenden

❌ Date format incorrect
✅ Lösung: ISO 8601 Format (2025-01-19T10:30:00+01:00)
```

### Problem: Sitemap wird nicht generiert

#### Debugging Steps:

```php
1. Rank Math → Status & Tools → System Status
2. PHP-Speicherlimit prüfen (min. 128MB)
3. WordPress-Permalinks aktualisieren
4. .htaccess Konflikte prüfen
5. Plugin-Konflikte deaktivieren
```

### Problem: Rankings sinken nach Rank Math Installation

#### Sofortmaßnahmen:

```php
1. NIEMALS mehrere SEO-Plugins gleichzeitig
2. 301-Redirects für geänderte URLs
3. Meta-Daten von altem Plugin importieren
4. Google Search Console auf Crawl-Errors prüfen
5. 4-6 Wochen für Ranking-Stabilisierung warten
```
